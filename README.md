# vue-mini-toast
```
简单的vue的弹窗插件（pop/alert/confirm/actionSheet）
```

## 安装
```
npm install vue-mini-toast -S
```

## 组件中使用
####pop
```
this.$pop(option);
```
#####option [Object]
|msg|align|theme|
|:-------:|:-------------:|:----------:|
|   居中   |     左对齐    |   右对齐   |
|=========|===============|============|
